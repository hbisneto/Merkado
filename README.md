# Merkado
Sistema simples de mercado com geração de cupom fiscal salvo em .txt

> Basta abrir o arquivo app.py e executar

Se for executar através do Terminal, navegue até o diretorio:

```
cd /Users/VOCE/Documents/Merkado
```
```
python3 app.py
```
